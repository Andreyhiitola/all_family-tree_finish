# Папка для фотографий

## Структура:
- `avatars/` — аватары людей (1.jpg, 2.jpg, ...)
- `gallery/` — галереи фото (1-1.jpg, 1-2.jpg, ...)
- `placeholder.svg` — заглушка для отсутствующих фото

## Использование:
В Excel колонка "Фото": `photos/avatars/5.jpg`
В Excel колонка "Галерея": `photos/gallery/5-1.jpg, photos/gallery/5-2.jpg`
